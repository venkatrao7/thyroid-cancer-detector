ThyroNet: A CNN Based Intelligent Tool For Thyroid Cancer Detection

About ThyroNet

ThyroNet is an advanced CNN-based tool designed to help everyone from patients to healthcare providers detect thyroid cancer from ultrasound images. It offers fast, accurate, and easy-to-understand predictions.

You can also ask queries related to thyroid cancer directly through the system for additional support and information.

Simply upload an ultrasound image on the home page. The system analyzes the image using deep learning models and instantly gives a prediction:

  • Green indicates a benign (non-cancerous) result.
  • Red indicates a malignant (cancerous) result.

This clear color coding helps users quickly understand their results and take the next steps confidently.

Features

  • AI-powered thyroid cancer prediction
  • Ultrasound image analysis
  • Fast and accurate classification
  • Interactive chatbot support
  • User-friendly interface

Technology Used

ThyroNet is developed using Convolutional Neural Networks (CNN), Flask, HTML, CSS, JavaScript, and Python technologies.
